Alpha Diversity

For each sample a single measure can be calculated describing the diversity within the sample. The most intuitive is how many different bacteria (OTUs) is observed. But there are others. The phyloseq package supports some X different versions.

References

The chapters ## to XX in The very good book that Dennis or Lukasz might know the name of (which is probably not the one in the link!) gives a description of alpha diversity.

Exercise

  • Calculate the alpha diversity for the XX samles using phyloseq.
library(phyloseq)
pp <- plot_richness(phyX)
AlphaD <- pp$data
  • Merge these with the meta data and make a visualisation of the distribution.
library(ggplot2)
AlphaD <- merge(AlphaD,metaD,by.x = 'samples',by.y = 'X.SampleID')
ggplot(data = AlphaD,aes(x = value)) + geom_histogram() + 
  facet_wrap(~variable, scales = 'free',ncol = 2)
